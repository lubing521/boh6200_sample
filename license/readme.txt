Purpose: to demonstrate how to get/update license file and show DIE ID value
Command: sample_license [-r] [-s input_license_file] [-g output_license_file]
Parameter:
          --read/-r: show DIE ID value in chipset with console
          --set/-s : update license file in flash
          --get/-g : get license file in flash
          --help/-h: used to display help information and exit.

Notice: The parameters are case-sensitive.
